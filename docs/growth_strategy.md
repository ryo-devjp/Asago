# Asago 成長戦略 — 機能提案 & 課金設計

> 想定ユーザー: ADHD・グレーゾーンなど時間管理が苦手で遅刻が多い学生・社会人
> コア課題: ① やるべきことが頭に入っていない ② 今やっていることに集中して時間経過に気づかない

---

## 1. 機能提案（優先度順）

### 凡例

| 項目       | 意味                                               |
| ---------- | -------------------------------------------------- |
| 優先度     | S（必須）/ A（高）/ B（中）/ C（低）               |
| 実装コスト | 小（〜1週間）/ 中（1〜3週間）/ 大（1ヶ月超）       |
| DL 効果    | ◎ 大きな差別化 / ○ プラス / △ 継続率向上（間接的） |
| 継続率効果 | ◎ / ○ / △                                          |

---

### ① 強制フルスクリーン通知 & バイブ連打（タイムオーバーアラート）

| 項目       | 値    |
| ---------- | ----- |
| 優先度     | **S** |
| 実装コスト | 小    |
| DL 効果    | ◎     |
| 継続率効果 | ◎     |

**概要:** 各タスクの目安時間を超過した瞬間、画面全体を赤くフラッシュさせ、バイブレーションを連打して「いま時間が過ぎている」ことを身体で感じさせる。ロック画面でも通知音+バイブで割り込む。

**なぜ効くか:** ADHDユーザー最大の敵「過集中による時間盲目」を物理的に断ち切る。普通のリマインダーは「通知を見ない」「気づかない」で終わるが、バイブ連打＋全画面赤フラッシュは無視できない。**他アプリにない最大の差別化ポイント。**

---

### ② ルーティンテンプレート（朝の支度プリセット）

| 項目       | 値    |
| ---------- | ----- |
| 優先度     | **S** |
| 実装コスト | 小    |
| DL 効果    | ◎     |
| 継続率効果 | ◎     |

**概要:** 「朝の身支度」「通勤前」「登校前」などのルーティンテンプレートをプリセットで用意。初回起動時にテンプレートを選ぶだけで即使い始められる。ユーザー自身のカスタムテンプレートも保存可能。

**なぜ効くか:** ADHDユーザーは「タスクを自分で洗い出して登録する」こと自体がハードル。テンプレートを選ぶだけにすれば初回離脱を大幅に減らせる。

---

### ③ タスクごとのタイマー自動進行 & 音声読み上げ

| 項目       | 値    |
| ---------- | ----- |
| 優先度     | **S** |
| 実装コスト | 中    |
| DL 効果    | ◎     |
| 継続率効果 | ◎     |

**概要:** タスクを上から順にタイマーが自動進行。「歯磨き 3分」が終わったら音声で「次は着替え、5分です」と読み上げ、自動で次のタイマーに進む。手動操作不要。

**なぜ効くか:**

- 「次に何をするか考える」という認知負荷をゼロにする
- スマホを見なくても音声で指示が来る → 過集中を断ち切れる
- **Routinery 等の競合と正面から戦える最低限の機能**

---

### ④ 曜日別ルーティン設定

| 項目       | 値    |
| ---------- | ----- |
| 優先度     | **A** |
| 実装コスト | 小    |
| DL 効果    | ○     |
| 継続率効果 | ◎     |

**概要:** 平日と休日、曜日ごとに異なるルーティン・出発時刻を設定できる。

**なぜ効くか:** 毎日同じ時間に出発する人は少ない。曜日別設定がないと毎回手動で変える必要があり、面倒で離脱する。

---

### ⑤ 連続遅刻ゼロ記録（ストリーク & ゲーミフィケーション）

| 項目       | 値    |
| ---------- | ----- |
| 優先度     | **A** |
| 実装コスト | 小    |
| DL 効果    | ○     |
| 継続率効果 | ◎     |

**概要:** 出発時刻までに全タスクを完了した日を「遅刻ゼロ」としてカウント。連続記録（ストリーク）を表示。ウィジェットにも出せる。

**なぜ効くか:** ADHDユーザーはドーパミン駆動型。「記録を途切れさせたくない」というモチベーションが継続利用に直結する。App Storeのスクショ映えにもなり、DLにも貢献。

---

### ⑥ iOS ウィジェット（ロック画面 & ホーム画面）

| 項目       | 値    |
| ---------- | ----- |
| 優先度     | **A** |
| 実装コスト | 中    |
| DL 効果    | ◎     |
| 継続率効果 | ◎     |

**概要:** ロック画面に「あと◯分」とカウントダウンを表示。ホーム画面ウィジェットには次のタスクとストリークを表示。

**なぜ効くか:** アプリを開かなくても残り時間が目に入る。ADHDユーザーは「アプリを開くこと自体を忘れる」ので、受動的に情報が届く仕組みが必須。**App Store での訴求力も高い。**

---

### ⑦ Apple Watch 連携（手首バイブ通知）

| 項目       | 値    |
| ---------- | ----- |
| 優先度     | **A** |
| 実装コスト | 大    |
| DL 効果    | ◎     |
| 継続率効果 | ◎     |

**概要:** Apple Watch に次のタスク・残り時間を表示し、タスク切り替え時に手首にバイブレーション。

**なぜ効くか:** スマホは別の部屋にあっても、Apple Watch は常に身につけている。手首振動は過集中を断ち切るのに最も効果的。ADHD界隈で「このアプリのためにApple Watch買った」と言わせたら勝ち。

---

### ⑧ 実績データ & 振り返り（所要時間の学習）

| 項目       | 値    |
| ---------- | ----- |
| 優先度     | **B** |
| 実装コスト | 中    |
| DL 効果    | ○     |
| 継続率効果 | ○     |

**概要:** 各タスクの実際にかかった時間を記録し、週次・月次で「あなたの歯磨きの平均は4分32秒です」とフィードバック。目安時間の自動調整提案も行う。

**なぜ効くか:** ADHDユーザーは自分の行動にかかる時間を慢性的に過小評価する（計画の錯誤）。実データで「実は5分じゃなくて8分かかってる」と気づかせることが遅刻改善の根本解決になる。

---

### ⑨ 家族・パートナー共有モード（見守り通知）

| 項目       | 値    |
| ---------- | ----- |
| 優先度     | **B** |
| 実装コスト | 大    |
| DL 効果    | ○     |
| 継続率効果 | ○     |

**概要:** 家族やパートナーに「あと10分で出発」「まだ着替えが終わっていません」などの状況を自動共有。見守る側もアプリをDL。

**なぜ効くか:** ADHDの子を持つ親、ADHD のパートナーがいる人は非常に多い。「見守る側」がインストールすることでDL数が2倍になるバイラルループを作れる。

### ⑩ 環境音 / BGM タイマー（集中 & 切り替え）

| 項目       | 値    |
| ---------- | ----- |
| 優先度     | **B** |
| 実装コスト | 中    |
| DL 効果    | △     |
| 継続率効果 | ○     |

**概要:** タスク実行中にホワイトノイズや環境音を流し、タスク切り替え時に明確な効果音を鳴らす。

**なぜ効くか:** 音の変化が「場面の切り替わり」を脳に知らせる。ADHDユーザーは視覚情報より聴覚情報のほうが行動の切り替えに効くケースが多い。

| ---------- | ----- |

**概要:** テーマカラーやアプリアイコンを複数から選べるようにする。

### ⑬ 天気 / 交通情報連携（出発時刻の自動調整）

| 項目       | 値    |
| ---------- | ----- |
| 優先度     | **C** |
| 実装コスト | 大    |
| DL 効果    | ○     |
| 継続率効果 | ○     |

**概要:** 天気予報（雨→傘タスク自動追加）、電車遅延情報を取得し、出発時刻を自動で前倒し。

**なぜ効くか:** 「雨の日は遅刻する」「電車遅延を計算できない」はADHD あるある。自動化すれば認知負荷ゼロで遅刻回避できる。ASO でも「天気連動」は刺さるワード。

## 2. 優先度マトリクス

```
                高い DL 効果
                    ↑
          ①強制通知  │  ③音声自動進行
          ②テンプレ  │  ⑥ウィジェット
          ⑤ストリーク│  ⑦Apple Watch
                    │
     ───────────────┼────────────────→ 高い実装コスト
                    │
          ④曜日別   │  ⑨共有モード
          ⑪Siri    │  ⑧振り返り
          ⑫外観     │  ⑩BGM
                    │  ⑬天気連携
                低い DL 効果
```

### 推奨実装順序

| Phase                  | 機能                                    | 目的                       |
| ---------------------- | --------------------------------------- | -------------------------- |
| **Phase 1（MVP強化）** | ①強制通知 ②テンプレート ④曜日別         | 最小コストで差別化を確立   |
| **Phase 2（定着）**    | ③音声自動進行 ⑤ストリーク ⑥ウィジェット | 継続率を上げてレビュー獲得 |
| **Phase 3（拡張）**    | ⑦Apple Watch ⑧振り返り ⑪Siri            | 課金転換率の向上           |
| **Phase 4（成長）**    | ⑨共有モード ⑩BGM ⑫外観 ⑬天気            | DLバイラル & 課金追加要素  |

### 課金設計の狙い

| 観点                | 設計意図                                                                                                                             |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| **Free → Pro の壁** | 「タスク5件」と「曜日別なし」が日常利用のペインになり、自然にPro検討へ。強制通知1日1回制限も「もっと欲しい」を生む                   |
| **Pro → Max の壁**  | Apple Watch・家族共有・天気連携など「なくても困らないが、あると生活が変わる」をMaxに集約。ヘビーユーザーの支払い意欲を吸収           |
| **Free の価値**     | Free でも「カウントダウン＋5件タスク＋1テンプレ＋週7ストリーク」で十分に遅刻改善を体験できる。レビューで★5を書けるレベルの体験を担保 |
| **年額割引**        | 月額の33%割引で年額に誘導。ADHD ユーザーは「忘れて解約できない」リスクがあるため、年額は十分な割引を提供して納得感を持たせる         |

## 4. ASO & マーケティング提案

| 施策                   | 内容                                                                                                              |
| ---------------------- | ----------------------------------------------------------------------------------------------------------------- |
| **App Store タイトル** | 「Asago - ADHDの朝を変えるルーティンタイマー」                                                                    |
| **キーワード**         | ADHD, 遅刻, 朝, ルーティン, タイマー, 支度, 時間管理, 発達障害                                                    |
| **スクショ訴求**       | ①「もう遅刻しない」②ストリーク画面 ③ Apple Watch ④ ウィジェット                                                   |
| **SNS 戦略**           | ADHD 当事者コミュニティ（X / TikTok）でのストーリー発信。「遅刻魔だった私が30日連続遅刻ゼロ」の実体験が最も刺さる |
| **レビュー誘導**       | ストリーク7日達成時にレビュー依頼ダイアログ（成功体験直後が最もレビュー率が高い）                                 |
